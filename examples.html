Max Date Range Example

This example only allows dates to be a max of a month apart.

If the end date is more than a month past the start date when selecting the start date, the end date is set to a months time

If the start date is more than a month before the end date when selecting the end date, the start date is set to a month ago

Uses the date.js config settings
$('#booking').daterangepicker( { dateRange: { months: 1 } });
